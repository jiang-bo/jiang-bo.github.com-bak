优化rails New时bundle Install速度

Dec 17th, 2011 | Comments

最近痴迷ruby和ror，但是每次rails new xxx时总会在bundle install 停留很久，学习了下，发现rails new时会进行bundle install，自动到rubygems.org上检查是否有更新。如果确定不需要更新，事实上是可以跳过这个步骤的，方法很简单： rails new my_app --skip-bundle 另外在单独执行bundle install时默认也会自动检查所有更新，解决方法时 bundle insall --local
